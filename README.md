# 🕌 NurChat - Chatbot Islami

Chatbot berbasis AI untuk membantu belajar agama Islam dengan aman dan ramah.

## ✨ Fitur
- Tanya jawab seputar Islam (shalat, doa, Al-Qur'an, akhlak)
- Filter keamanan untuk konten tidak pantas
- Tampilan Islami yang menarik

## 🚀 Cara Menjalankan
```bash
pip install streamlit google-generativeai
streamlit run chatbot_islam.py
